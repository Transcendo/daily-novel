# 死人账户

母亲已经死了三年，为什么她的医保缴费单今天会从自助取药机里吐出来？

林槐站在药房窗口里，手里还捏着一板刚扫码的降压药。玻璃外的雨把上午压得发灰，取药大厅里挤满了湿伞和塑料袋。机器“咔哒”一声，又吐出半截热敏纸，像没吐干净的舌头。

她先看见姓名：陈桂英。再往下，是慢阻肺雾化药，两盒，支付时间九点二十七。

陈桂英是她母亲。

三号窗口的病人敲玻璃：“药师，轮到我没有？我车还停外头呢。”

林槐把单据翻过去压在处方夹下，声音照旧：“下一个，取药码。”

大厅另一头，医保稽核员的蓝色雨衣还没脱。他把平板搁在咨询台上，对主任说：“今天只查异常账户。死人账户、重复报销、同日多点取药，系统都弹出来了。先从自助机开始。”

林槐的手指在扫码枪上滑了一下，枪头扫空，发出刺耳的“滴——”。

取药机又响。第二张缴费单滑出来，还是陈桂英。

这回不止她看见。保安老周从门口冲过来，鞋底带着一串水。他比平时更快，没先喊“伞收一下”，而是一把按住出纸口。

“机器坏了。”老周笑着对排队的人说，“别围，别围，我来处理。”

他手背上青筋鼓起来，把那张单据揉进掌心。

林槐隔着玻璃看他。老周也看她，眼神不是求，是警告。他把湿雨衣往身前拢了拢，里面鼓出一个方形包。

稽核员已经走到自助机旁：“谁动了单据？”

老周说：“热敏纸卡住，我怕病人吵。”

“卡纸不用撕。”稽核员伸手，“给我。”

排队的人开始抱怨。一个抱着孩子的女人说药快断了；一个老人把医保卡拍在窗口台面上，说自己喘不上来；主任从后面探头，低声叫林槐：“先把窗口稳住，异常账号你等下配合。”

林槐把陈桂英的第一张单据从处方夹下抽出来。纸还热，母亲的名字黑得像刚写上去。系统屏幕右上角弹出红框：账户状态异常，请人工复核。

她点开历史记录。

三年前，两次续药签名跳出来：林槐。

那是母亲最后一个冬天。她瘦得只剩一件棉袄的重量，咳一夜，第二天又不肯排队。林槐替她续过两次药，流程上不合规，她知道。那时她想，反正人还活着，反正只是母亲，反正不会有人追。

现在红框把“反正”两个字撕开了。

老周退到窗口边，把揉皱的单据塞进林槐面前的小孔：“小林，机器乱吐的，撕了。你妈的事，别再让人翻。”

林槐没接。

稽核员转过来：“你认识这个账户？”

窗口外一下安静。雨声贴着门廊往下砸。

老周抢先说：“社区里重名多，陈桂英又不稀奇。”

稽核员看平板：“身份证后四位 2147。死亡注销时间，三年前十二月。”

抱孩子的女人急了：“死人卡还能取药？那我们排半天算什么？”

老周脸涨红，冲她吼：“你取你的药，少掺和！”

孩子被吓哭。主任的脸沉下来：“老周，你注意影响。”

老周像被这句话抽了一下。他往后退，雨衣里的方包碰到玻璃，发出塑料袋摩擦声。林槐看见袋口露出一角药盒，正是雾化药。

稽核员也看见了：“你雨衣里是什么？”

“我的。”老周说。

“拿出来。”

“我自己买的。”

“拿出来。”

老周没动。他看了林槐一眼，嘴唇抖着，声音压得很低：“小林，我儿子今天来接我。他不知道我还在这儿干外包保安。他以为我在物业办公室坐班。你别把事闹大。”

林槐终于明白他怕的不是一张单。他怕自己在儿子面前从“还有体面的人”变成偷医保药的人。

稽核员伸手去拿他的雨衣。老周猛地一挡，手肘撞到叫号屏，屏幕闪了两下，排队号停住。大厅里彻底炸开：“怎么不叫号了？”“我药呢？”“医院死人都能取，我们活人取不到？”

主任喊林槐：“先手工发药！”

林槐却从座位上站起来。她拿起柜台内的异常账户核查单，走出窗口。门禁响了一声，玻璃门在她身后合上，她第一次站到队伍和机器中间。

稽核员问：“你是药房药师林槐？”

“是。”

“这个账户你是否认识？”

“认识。陈桂英是我母亲，三年前去世。”

老周闭了闭眼。

稽核员继续问：“历史记录里有两次你本人的续药签名，是否属实？”

主任在后面急得叫她：“林槐，话想清楚再说。”

林槐看着那张纸。母亲的名字在雨天里像一块没埋好的骨头。她忽然发现，真正让她怕的不是老周用亡母账户，而是这张单据证明母亲死后，她还把自己藏在“当时没办法”里。

她说：“属实。我违规替她续过药。今天这两盒不是我取的，但这个账户异常，我申请现场冻结。”

她在核查单上写下陈桂英的身份证号，又写自己的工号。笔尖停在“经办人签字”前一格，主任的手机响了，窗口病人还在催，老周像要冲过来夺纸。

林槐没有抬头，把名字签完。

签字一落，稽核员抽走核查单，按下平板确认。取药机屏幕跳出一行字：账户已冻结，请至人工窗口处理。

老周像被抽掉骨头，慢慢解开雨衣。他从里面拿出那只药袋，放到咨询台上。袋子湿了一半，处方联被他攥得皱巴巴的，上面不是陈桂英的名字，而是另一个老人，欠费，待补缴。

“她老伴断药了。”老周哑着嗓子说，“我先拿，下午补钱。上回有人教我，说死人账户还挂着，能过机。我就……我就想撑一次。”

没人接他的话。大厅外，一个年轻男人撑伞站在门口，朝里张望，喊了一声：“爸？”

老周没有回头。

稽核员收走林槐胸前的工牌：“你先暂停窗口，跟我去办公室做笔录。”

林槐点头。她走到人工窗口，把那位喘不上气的老人扶到椅子上，对主任说：“按急用临时处方走，费用先从我个人账户垫，记录写清楚。”

主任看着她，半晌没说话，最后把处方章拍在台面上。

叫号屏重新亮起：“请 A103 到三号窗口。”

林槐没有再回窗口。她把自己的工牌放进稽核员的文件袋里，听见封口贴“刺啦”一声合上。那声音很轻，却比雨声清楚。

母亲的缴费单为什么会吐出来，她现在知道了：不是死人回来取药，是活人一直借死人躲过该承担的账。今天机器把名字吐出来，她就不能再把自己吞回去。
