# 工牌别藏回去

雨从午后一直下到傍晚。旧缝纫室的窗缝里钻进潮气，墙上的广播喇叭先咳了一声，又没了动静。

林蔚把半湿的校服袖口压到缝纫机压脚下，问：“这里面夹的什么？”

许苗站在桌边，头发梢还滴水，眼皮因为困倦垂着，听见这句，手先往袖口上一按。

已经晚了。袖口里层开了线，一张小小的塑料工牌从夹缝里滑出来，磕在铁皮机台上，声音很轻，却像把门闩落了下来。

工牌被汗泡得发皱，透明壳边缘发白，上面印着便利店的绿色标识。名字那一栏被反复摩擦，只剩一个“苗”字还清楚。背面贴了一条窄窄的排班纸，红笔写着：23:00—07:00。

家长会还有二十分钟开始。走廊里已经有家长套着蓝色鞋套往高三一班走，鞋底擦过瓷砖，沙沙地响。雨伞尖上的水滴砸在门口的小塑料桶里，一下一下。

林蔚第一反应不是心疼，是恼火。

她今天刚把许苗排成“优秀学生代表”，还要在会上一开场讲严管示范班。许苗昨晚晚自习又睡过去，语文卷子空了两道古诗默写。林蔚原本准备在家长会上轻轻点她一句：越到临门一脚，越不能拿前途开玩笑。

现在，这张夜班工牌从袖口里掉出来，像一个更难看的证据。

“你晚上去这里？”林蔚捏起工牌。

许苗把手背到身后，声音很低：“林老师，先缝衣服吧。”

“我问你，为什么藏在袖口里？”

许苗看了一眼门。门外有家长问厕所在哪儿，年级组长赵主任抱着保温杯在走廊尽头喊：“一班林老师，发言稿准备好没有？你第一个。”

许苗说：“我等会儿要上台，袖子破着不好看。”

这句话把林蔚堵住了。确实不好看。校服袖口翻着一条毛边，湿布贴着许苗细瘦的手腕，露出里面灰白的线头。胸前那枚“优秀学生代表”的胸牌还没有别，躺在发言稿上。发言稿第一页写着：严格是最朴素的负责。

林蔚把工牌放到稿纸边，重新踩下脚踏板。

老缝纫机是从劳技教室搬来的，平时兼给广播站堆杂物。脚踏板涩，踩下去要用小腿顶，机针上下两次，忽然卡住。线轴从立杆上跳下来，滚到桌脚，绕住了一只旧话筒线。

“别动。”林蔚弯腰去捡线轴。

许苗已经蹲下去了。她的动作快得像便利店听见进门提示音时抬头，手先伸过去，牙齿咬住线头，想把缠在话筒线上的白线扯开。

林蔚说：“我叫你别动。”

许苗松开牙，低头不说话。

林蔚把线重新穿过针孔。她这几年管班、开会、填表，什么都能赶在截止前做完，偏偏穿线这件小事让她眼睛发酸。雨声贴着玻璃，门外的人声越来越密。她踩下踏板，布料往前走了一截。

“好了。”她松了一口气。

许苗却盯着袖口，脸色白了一下。

校徽被她缝反了。原本应当朝外的校名倒扣在袖口内侧，蓝白相间的一块，歪歪地贴着。针脚急，间距忽长忽短，像一串没写完的批注。

林蔚的脸热起来：“拆掉重来。”

“我来。”许苗抢过拆线器。

她怕耽误时间，手指压着湿袖口，拆线器尖尖地挑进去。线被雨水浸软，不肯断，她用力一挑，尖头滑开，扎进左手食指。

血一下冒出来，不多，圆圆的一点。许苗本能地甩手，血擦到桌上的胸牌边缘，红色沿着透明壳渗开，沾住“优秀”两个字的一角。

两个人都停住。

广播麦克风忽然啸叫了一声，尖得让人牙酸。墙上喇叭里传来学生会同学试音：“各位家长，请到对应班级就座，家长会将在五分钟后开始——”

“创可贴呢？”林蔚拉开抽屉。

抽屉里只有钝掉的剪刀、半盒粉笔、两枚缺角纽扣和去年运动会剩下的号码布。许苗把手指往校服裤侧擦，被林蔚一把抓住。

“别擦。”

许苗的手很冷。指腹有细小裂口，裂口里嵌着洗不净的灰。林蔚这才闻到她袖子上的味道，不只是雨水，还有便利店关东煮汤底的甜咸味、茶叶蛋卤水味、塑料袋在热柜上烘久了的味道。那味道钻在湿布里，和旧缝纫室的机油味混在一起。

工牌背面的排班纸因为刚才被林蔚捏过，翘起一角。下面还有一行小字：本周替班三晚，住宿费自结。

林蔚看着那几个字，想起上个月交上来的住宿申请。许苗也交了。表格很薄，字写得小，家庭情况栏只填了“母亲务工，租住城西”。同时交来的还有另一个男生的材料，父亲病历、低保证复印件、班级成绩排名，都齐齐整整。林蔚那天赶着去区里汇报，拿红笔在更好看的那份材料上画了圈，把许苗那张压在档案袋后面，说等下一批。

下一批还没来。

她听见自己问：“你住宿费还差多少？”

许苗把受伤的手握成拳：“林老师，别让我妈知道。”

不是“帮帮我”，不是“我不容易”。她只说这一句，像把门从里面抵住。

门外赵主任又过来，保温杯盖子拧开，热气扑在玻璃门上。他没进来，只从门缝里探头：“林老师，横幅让雨泡皱了，我让总务拿胶带。你那个优秀学生——许什么？许萌？准备好了没有？第一个发言不能空着啊。”

许苗把工牌翻面，往发言稿下塞。

林蔚按住那张稿纸。

赵主任还在说：“你今天好好讲，区里照片也要拍。严管示范班嘛，得有样子。哎，外面有个家长找你，说是许同学妈妈。”

许苗的肩膀缩了一下。

门口响起一个女人的声音：“林老师，我家苗苗是不是又犯错了？”

那声音不高，带着小心。林蔚抬头，看见一个女人站在雨伞架边，裤脚湿了一圈，手里提着一个透明塑料袋。袋子里有两个凉馒头，外皮被水汽泡得发皱；还有一把零钱、几张便利店小票、一个舍不得扔的湿伞套。她一只手攥着袋口，一只手把鞋套往上提，提了两次都滑下来。

许苗立刻把受伤的手藏到身后。

“妈，你先去教室。”

女人没敢进门，先看林蔚的脸色，又看女儿湿透的袖口：“你衣服怎么了？我就说今早带伞，你非说不用。林老师，她是不是晚自习睡觉了？我回去说她。”

“妈。”许苗的声音压低，“你去教室。”

女人被这句顶得脸上发红，嘴唇动了动。她大概也怕丢脸，在老师门口站久了不好看，便把塑料袋往身后藏，像袋子里的凉馒头也会让人看不起。

赵主任夹在中间，看看表：“先入座，先入座。林老师，三分钟。”

林蔚本来可以点头，让许苗母亲先走，把会开完。会后再叫许苗谈，补一张申请，找个不难看的说法。她甚至可以把工牌暂时塞回袖口，校徽拆了重缝，胸牌擦干净。家长会照开，她的发言照读。严格，负责，冲刺，家校合力。每个词都已经在稿纸上排好队。

她松开稿纸，纸角因为潮气卷起，露出下面助学材料的复印清单。那是她刚才为了找胸牌一起带来的，准备会后提醒家长补资料。许苗那张表在最底下，右上角有她自己的红笔字：材料不够充分，暂缓。

林蔚忽然觉得旧缝纫室太小了。缝纫机、广播麦克风、发言稿、湿校服、工牌、胸牌，全挤在一张桌上。每一样都不大，却把她围住。

许苗伸手去拿工牌，声音几乎没有：“我妈不知道。我上夜班是在离她租屋远一点的店，她以为我住校。老师，我不影响高考，我就是有时候困。”

林蔚看着她的眼睛。许苗眼皮沉，眼白里有红丝，却不是撒谎的样子。她说“不影响高考”时很急，像不是为自己辩解，而是在替某个薄薄的东西挡雨。

“你为什么不早说？”林蔚问完就后悔了。

许苗低头，用拇指压住食指伤口：“说了也要交钱。”

这句很轻，比刚才麦克风的啸叫还让林蔚耳朵发麻。

门外广播正式响起：“高三年级家长会现在开始。请各班班主任到班级主持。”

赵主任在外面急了：“林老师！”

林蔚拿起胸牌，想用纸巾擦血。旧纸巾一沾水就起屑，越擦越糊。红点还在“优秀”旁边，擦不掉。许苗伸手要接：“别戴了，脏。”

“要戴。”林蔚说。

她把自己的深灰外套从椅背上取下来。外套袖口也被雨气打潮，内衬不算干。她原本穿着这件上台，显得稳重些。现在她把外套抖开，披到许苗肩上。许苗往后躲了一下。

“湿。”许苗说。

“比你校服干。”

林蔚第一次别胸牌，针从外套布料里顶不出来。她的手指因为刚才穿线和捡线轴有点僵，针尖找不到出口。赵主任推门进来半步，看见这一幕，话停在嘴边。

“林老师，流程——”

“我知道。”林蔚没抬头。

第二次，针尖终于穿过布料，却歪了，胸牌斜挂着。许苗想伸手扶，被林蔚按住。

“疼手。”

第三次，胸牌别稳了。红点还在，贴着“优秀学生代表”几个字。歪校徽没有拆完，一半针脚还反着，露在外套袖口下面。许苗低头看见，脸上露出一种难堪的表情，不像被表扬，倒像站在光里无处可藏。

林蔚拿起便利店工牌。许苗的手动了一下。

“林老师。”

“工牌别藏回去了。”

林蔚把工牌放进发言稿上面，又把稿纸合起来。她没有撕，手指攥住第一页，揉成一团。纸团里“严格是最朴素的负责”几个字皱在一起，看不清了。

走到门口时，许苗母亲还站在那里。她大概听见了几句，又不敢确认，手里的塑料袋被攥得吱吱响。凉馒头挤在袋角，像两块灰白的石头。

林蔚停下，对她说：“许苗妈，您一起进去吧。”

女人慌忙点头，又看见女儿肩上的外套和胸牌，眼里先是茫然，随后立刻去看许苗的脸：“苗苗，你手怎么了？”

许苗把手放到身侧，没回答。

赵主任跟在后面，压低声音：“林老师，今天区里——”

林蔚说：“赵主任，麦克风先给我。”

高三一班教室里坐满了家长。窗户开着一条缝，雨声和楼下操场上的积水声一起涌进来。黑板上写着“家校同心，决胜高考”，下面横幅果然被雨泡皱了一角，透明胶贴得不平。第一排有个父亲还在擦眼镜，有个母亲把保温饭盒放在膝盖上，怕汤漏出来。学生们站在后排，校服颜色一片潮湿的蓝白。

许苗跟在林蔚身后进门时，教室里安静了一瞬。有人看见她肩上的成人外套，有人看见她袖口歪着的校徽，有人看见胸牌边上那一点血。

许苗母亲站在门边，没立刻坐。她还提着那袋凉馒头，鞋套一只高一只低，像随时准备被老师叫出去。

林蔚站到讲台前，把揉皱的发言稿放在一边。麦克风底座有点松，她扶了一下，发出一声闷响。赵主任在门口把流程表按在保温杯下面，嘴唇抿紧，最后没有打断。

林蔚看着台下。她看见许苗坐在第二排边上，外套宽，袖子长，遮住半只手；她也看见那枚工牌安静地躺在讲台边，和沾血的胸牌隔着一支粉笔。许苗刚才进门前，还是把工牌往里侧推了推，不是藏，只是不让它掉下去。

林蔚开口时，声音比她想的哑。

“各位家长，今天原本我要讲我们班怎么严格管理。”

台下有人抬头，准备听熟悉的开场。

林蔚把那团发言稿拿起来，又放下。

“但我先承认一件我没看见的事。”

教室里彻底静了。雨点敲在窗台上，一粒一粒。

“许苗同学这段时间晚自习打瞌睡，我批评过她，也准备今天请家长一起提醒她。我没有先问清楚，她为什么困。”林蔚停了一下，“她的住宿申请，上个月交到我这里，被我压在后面。我用‘材料不够好看’四个字，把一个学生应该被看见的困难暂缓了。这是我的错。”

许苗猛地抬头。她脸上不是感激，是惊慌，像林蔚真的把一扇门推开了，里面还没收拾。

许苗母亲站在门边，塑料袋被她攥得更紧。袋子里的小票滑出来一张，落在鞋边。她弯腰去捡，捡到一半停住，手悬在半空，好像终于明白那些夜里女儿不在身边的空处，不是学校管得严，而是别的什么东西替她们撑着。

林蔚没有把便利店、夜班、时薪一件件说给所有人听。那不是她的资格。

她只说：“今天优秀学生代表还是许苗。胸牌刚才被我弄脏了，校徽也是我缝反的。她站在这里，不是因为从来没有出错，也不是因为我们管得多漂亮。是因为她在很多人没看见的地方，硬撑着把该做的事做完。”

后排学生里有人低下头。第一排擦眼镜的父亲把眼镜布停在手里。赵主任咳了一声，又把咳嗽咽下去，手掌按住那张流程表，纸角慢慢翘起来。

林蔚转过身，对许苗说：“许苗，请你到前面来。”

许苗没有立刻动。她看了母亲一眼。许苗母亲也看着她，嘴唇张了张，没叫“苗苗”，也没问“你怎么不说”。她只是把那袋凉馒头放到旁边空椅上，两只手空出来，像怕等会儿女儿走过来时，自己连接一下都接不好。

许苗站起来。外套从肩头滑了一点，她用没受伤的手扯住，往前走。经过母亲身边时，母亲伸手碰了碰她的袖口，碰到那块缝反的校徽，又缩回去。许苗没有躲。

她走到讲台边，低头看见工牌。那张被汗泡皱的小牌子在灯光下泛白，背面排班纸翘着一角。她伸手，把工牌往胸牌旁边摆正了一点。然后她把自己袖口上那半截歪校徽也慢慢翻出来，露给台下看。

林蔚把麦克风让给她，却没有逼她说。

许苗站了几秒，最后只说：“我会把今天的卷子补完。”

这句话太小，甚至不像优秀学生代表发言。可教室里没有人笑。

许苗母亲坐下时，凉馒头还在旁边椅子上。她没有哭，只把塑料袋口松开，又重新系好，这次没往身后藏。赵主任把流程表从保温杯下面抽出来，看了一眼，划掉第一项“优秀班主任经验介绍”，把笔帽咬在嘴里，半天没找到替换的词。

林蔚站在讲台边，忽然觉得肩膀有点冷。外套在许苗身上，袖口湿，胸牌带血，校徽还歪着。她知道明天区里照片不会好看，评优材料也要重写。她还知道，许苗的住宿申请不能只用一句“我会问问”搁过去。会后她要把压在档案袋底下的那张表拿出来，当着许苗和她母亲的面重新补，问缺什么，缺的由她去找，不再让一个十七岁的孩子用夜班把空白填上。

但这些还在后面。

现在，家长会已经被第一句话改掉了。那团发言稿躺在讲台角落，便利店工牌没有回到袖口里。它和沾血的胸牌并排放着，像两枚不配套却都是真的证件。

雨还在下。旧校徽反着，许苗用指尖轻轻压了压，没有再藏。
