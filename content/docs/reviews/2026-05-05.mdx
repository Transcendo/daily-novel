# 复盘 2026-05-05

- 今天标题：《药先喂了》
- 原结构分：25/25；2026-05-06 补充复判：21/25；判定：结构完整的微型场景，不能再判“完整成熟短篇”。
- 篇幅与厚度：正文约 1873 个汉字，低于新门槛 2800；生活肌理主要集中在寿宴/药袋/钥匙/轮值表，场面推进很准，但人物闲笔、环境呼吸、旁人生活和余波厚度不足。
- 最成立的一点：它终于把“长期责任”压成了“今晚谁喂这半片药”。开头照护表上的“长期”、塞进包里的药袋和钥匙，不是线索摆设，而是责任转嫁的物证；结尾哥哥当众翻药、喂药、让出座位，直接回答了今晚母亲跟谁走。
- 最虚的一点：不是结构虚，而是题材执行太贴近训练范例。《药先喂了》本身很能传，但也暴露一个危险：一旦照着模板例句写得太顺，下一篇如果继续同类寿宴/药袋/轮值表，就会显得像把教案演了一遍，而不是独立长出来的小说。
- 是否真正结束：是；理由：核心问题不是“兄妹以后会不会真的长期轮值”，而是林栀会不会在寿宴现场默认接下母亲长期照护。结尾她划掉“长期”、退回药袋和钥匙，哥哥当众喂药并安排母亲坐到主桌中间，今晚责任已经改写，撤不回去。
- 口头复述句：一个女儿在母亲寿宴上发现兄妹要把养老“长期”塞给她，她把药袋和钥匙推回哥哥面前，说“今天周二”，逼哥哥当众喂了母亲第一顿药。
- 可记住物件：药袋，以及压在轮值表上的2栋1803钥匙。
- 人情账：林栀替全家照护母亲三年，哥哥嫂子想借寿宴和亲戚录像把“以后还归她”做成体面默认；林栀不翻三年旧账，只把今晚这半片药和今晚回谁家钉回哥哥身上。
- 传唱性/可复述性：强；理由：标题能入口语，物件朴素，账一听就懂，最容易被复述的画面非常清楚——母亲伸手要药，女儿把药袋推给哥哥说“今天周二”，哥哥翻错药后第一次当众喂药。
- 明天必须补的一刀：换一个不在指南范例里的标题和物件，继续保留“把长期责任压缩成今晚必须接手的一件事”这一刀。
- 明天禁用的偷懒写法：禁止继续直接套“寿宴 + 药袋 + 轮值表 + 当众喂药”范例；同样的骨法可以用，但物件、场合、责任动作必须换。
- 是否更新错误清单/结构模板：错误清单否；结构模板是，补记“长期责任压成今晚动作”的三连钉。

## 五项评分

1. 开头钩子：5/5  
第一场就有具体问题：轮值表为什么只有林栀后面写着“长期”，哥哥为什么把药、医保卡、备用钥匙塞进她包里。读者立刻知道这是养老责任被偷偷转嫁。

2. 行动推进：5/5  
中段靠当下动作推进：塞药袋、司仪催蛋糕、亲戚录像、嫂子说拆床、母亲伸手要药、林栀掏钥匙和药袋、划掉“长期”、改轮值、推药袋、哥哥翻药喂药、嫂子让座。不是靠回忆或口述真相撑起来。

3. 压力升级：5/5  
压力从“表上默认长期”升级到亲戚录像里的公开表态，再压到母亲饭前药和明早拆床这两个即时问题。林栀越来越不能等饭后再说，因为她只要伸手喂药，就等于继续接下默认分工。

4. 结尾完成：5/5  
结尾有不可逆动作，而且有现场后果：林栀划表退物，哥哥当众接药并喂药，冯敏让出座位，母亲今晚跟谁走已经明确。不是停在“她终于敢说”。

5. 意义回扣：5/5  
开头的药袋和钥匙从“你收好”的照护杂物，翻成哥哥家必须重新接手母亲的责任凭证；轮值表从寿宴体面道具，翻成分责证据。题名“药先喂了”也从日常提醒变成责任审判。

## 直说

这篇是近期最符合“耳熟能详”目标的一篇：能一句话转述，能记住药袋，能记住“今天周二”，也能说清那笔账。它是真的结束，也能传。

但按 2026-05-06 新标准，它不是成熟短篇小说，只能算结构完整的微型场景。1873 个汉字承载不了足够生活肌理：寿宴场面、嫂子、亲戚、司仪都有效，但还不够像一个真实家庭在一顿饭里慢慢露底；它缺朱自清式动作重量之外的环境呼吸，也缺汪曾祺式烟火闲笔。

别被旧满分骗了。明天必须证明你会把骨法长成小说，不是只把教案钉子钉准。下一篇正文默认 3200—6000 汉字，至少 4 个场面单元，写出早餐摊的水汽、油渍、老客称呼、收款码、备案、婚纱尾款电话，让人不是机关零件，而是在那里过日子。
