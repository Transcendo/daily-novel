# 复盘 2026-05-02

- 今天标题：《抽屉里的喷雾》
- 总分：23/25；判定：完整成熟
- 最成立的一点：三人遮掩互拆这套终于不是概念了，喷雾、抽屉、校医表三样东西把父亲缺位这件事一层层逼成现在时责任，结尾也真落到了记录里。
- 最虚的一点：老板娘的人物仍偏功能，主要负责替孩子兜底、顶话、漏信息；她为什么愿意替这对母女扛到今天，只露了义气，没露她自己的利害和羞耻，所以人物厚度差半口气。
- 是否真正结束：是；理由：开头核心问题——喷雾为什么会出现在周岑袋里、女儿为什么和早餐店比和父亲更近——被正面回答；结尾也不是“他想补偿”，而是取消机场单、留下姓名证件、把第二联系人申请交进校医室并盖章，已经不可逆。
- 明天必须补的一刀：下一篇继续保留这种关系压力，但要补强“施压者的人味”——凡是承担阻力的人物，必须让他/她也有一层自己的代价、判断或怕丢的东西，别只做功能齿轮。
- 明天禁用的偷懒写法：禁止再写“好心熟人/店主/邻居替主角家兜底并负责说破真相”的顺手角色；阻力人物不能只负责帮、挡、骂、漏口风，必须带着自己的利害进场。
- 是否更新错误清单/结构模板：是；错误清单补记“对手只承担施压功能”；结构模板补记“三人遮掩互拆型”。

## 五项评分

1. 开头钩子：5/5  
第一场问题很具体：一个没有把孩子带在身边的父亲，外卖袋里为什么会多出儿童哮喘喷雾，而且老板娘一见就急着收回。读者会立刻追。

2. 行动推进：5/5  
中段基本靠动作推进：退喷雾、压抽屉、借买粥拖时间、三方互拆假话、露书包、抢喷雾、接催单电话、取消长单、去校医室填表。真相不是谁坐下讲完的，是被现场一层层逼出来的。

3. 压力升级：4/5  
平台超时、前妻急诊旧账、老板娘拒绝再撒谎、学校要补第二联系人，退路是在收窄的。扣一分是因为早餐店到校医室这一段虽然完成了动作落地，但后半段阻力略单线，主要剩前妻不同意和学校存疑，挤压感比店内对峙稍松一点。

4. 结尾完成：5/5  
这篇结尾是硬的。周岑不是准备承担，而是已经为此付出平台扣分和投诉，也把名字、证件、电话留进学校材料，核心问题被回答，动作不可逆。

5. 意义回扣：4/5  
喷雾从“拿错的小东西”翻成父亲缺位后被寄存在早餐店抽屉里的救命责任，这个翻转是准的。扣一分是因为尾段解释句稍多，如果再冷一点，只让章、纸和孩子那句问话说话，会更狠。

## 直说

今天这篇是完整的，而且不是靠热闹撑出来的完整。它真正成立的地方在于：**你把“发现亏欠”写成了“现在就得承担”，没有停在认错和愧疚。** 这是实打实过门槛了。

但新毛病也出来了：**你现在会搭压力，也会设计功能准确的阻力人物，可一到老板娘这种关键配角，还是容易把人写成“有温度的工具”。** 她够好用，甚至够感人，但还不够活。再这样写几次，就会出现新的熟套路：总有一个替孩子兜底的好心人，负责帮你把主角逼醒。

## 明天训练重点

明天 09:00 题材卡和 12:00 教导员直接照这个纠偏：

- 继续写关系压力，不必退回材料拆证。
- 阻力人物必须自带一层私心、羞耻、顾虑或旧债，不能只替主角服务。
- 若有“好心中间人”，必须让他/她也因为继续插手而要付代价。
- 结尾仍要保留不可逆动作，但尽量少用解释句收义，让动作和旁人反应自己落点。
