# 复盘 2026-05-03

- 今天标题：《死人账户》
- 总分：22/25；判定：完整成熟
- 最成立的一点：开头钩子很硬，死人账户当场吐单，稽核员就在场，问题具体，压力立刻成形；后面也基本靠抢单、藏药、停号、出窗口、签核查单这些动作推进，不是坐下来讲旧事。
- 最虚的一点：结尾最后一段解释得太满，把已经成立的动作又说了一遍；另外老周的人味比前一篇有补，但还是偏承担结构功能，主要负责藏、抢、求、塌，个人判断还差一层更反咬人的复杂性。
- 是否真正结束：是；理由：开头核心问题——亡母账户为什么还在出药、是谁在用——被正面回答；结尾也不是“她想承担”，而是她现场承认旧违规、申请冻结、签字、交出工牌，后果已经启动。
- 明天必须补的一刀：下一篇要硬练“少解释一句，后果多落半步”。结尾动作之后，尽量让回应、沉默、损失自己说话，不要再补一段主题说明替读者总结。
- 明天禁用的偷懒写法：禁止再用最后一段把全篇意义讲明白；也禁止再写“主角签字/上报后，再用一段内心总结完成主题”。
- 是否更新错误清单/结构模板：已更新错误清单；结构模板本次不新增。

## 五项评分

1. 开头钩子：5/5  
第一场问题非常具体：死了三年的母亲，为什么今天还从医院机器里吐出缴费单，而且稽核员就在旁边查死人账户。读者会立刻追。

2. 行动推进：5/5  
中段主要靠当下动作推进：压单、继续发药、老周抢纸、藏药袋、稽核员逼问、叫号屏被撞停、林槐走出窗口、签核查单、冻结账户、交工牌。真相基本是被现场动作和程序压力逼出来的。

3. 压力升级：4/5  
制度压力、窗口压力、自我牵连、老周年轻儿子在门口这层羞耻压力都有效，局势也一直在收窄。扣一分是因为老周真正为什么铤而走险，直到很后面才落清，前半段更强的是功能性紧张，不是人物之间互相啃咬的复杂压力。

4. 结尾完成：4/5  
不可逆动作是有的：签字、冻结、暂停窗口、交出工牌，核心问题也被回答，所以整体完成。扣一分是因为最后又补了一段解释性总结，削弱了动作本身的硬度，像作者怕读者没懂，主动扶了一把。

5. 意义回扣：4/5  
“缴费单吐出来”从系统异常翻成“活人借死人躲账”，也连回林槐自己过去那两次违规续药，意义翻转成立。扣一分仍在于尾段点义偏直，少说一点会更狠。

## 直说

今天这篇是完整的，而且不是侥幸完整。你把“旧账户异常”写成了真正会咬回主角的现在时后果，门槛是跨过去了。

但也得直接说：这篇最露怯的地方不是结构，而是你又有点不放心动作，结尾忍不住替它解释。前面已经够硬了，最后那一段一出来，锋利度就往回缩了一点。还有，最近三篇虽然题材不同，但你越来越熟练的一条路径已经很明显：**异常暴露—现场程序压力—角色当场签字/留名承担。** 这条现在能成，但再连用，很容易手艺化。
